<h1>Card Study Timer</h1>
A timer based on an internet trend where you pick a random card and study one subject for that amount of time.

<h2>How it works</h2>
On clicking start, a card is randomly selected and a timer starts based on that time. Once the time runs out, or stop is pressed, the start button can be pressed again to be reused.

<h2>The cards</h2>
<ul>
  <li>Ace - study for 10 minutes</li>
  <li>2-10 - study for 10x the number on the card</li>
  <li>Jack - break for10 minutes</li>
  <li>Queen break for 15 minutes</li>
  <li>King break for 20 minutes</li>
  <li>Joker picks a different card, and doubles it</li>
</ul>
